Privacy Policy

We respect your privacy and are committed to protecting it. This Privacy Policy outlines how we handle information in connection with the use of our app. Our primary goal is to make sure you understand that we do not collect or store any personal data.

1. Information We Collect

We do not collect or store any personal information. We do not ask for or track your name, email address, location, or any other personally identifiable information (PII).
The only data we collect are general usage statistics, such as the number of API requests and the time of day these requests are made, to help us improve our app’s performance.

2. How We Collect Usage Data

We use third-party services like Google Cloud Run to host the backend API that powers the app's route calculation features. Built into the Google Cloud Run product are basic analytics that we use to monitor and analyze usage patterns for our API. This data includes:

  • The number of requests made to the API
  • The time of day the requests are made

This information is fully anonymous and cannot be traced back to individual users.

3. How We Use Usage Data

We use the anonymized usage data to:

  • Improve the performance and functionality of our app
  • Understand usage trends to ensure availability

We do not share, sell, or disclose this data to any third parties except as required by law.

4. Cookies and Tracking Technologies

Our app and website do not use cookies, tracking pixels, or any similar technology to collect information.

5. Third-Party Services

As stated before, our API is hosted on Google Cloud Run. We use its built-in tools for tracking API usage trends for the purposes mentioned above. Google Cloud Run simply provides a platform for the API code to run on. Google does not have access to any personal information (since we don't collect any), and their privacy policies can be viewed on the websites for their respective products.

6. Data Security

We do not collect any personal data. Thus, there are no personal data security concerns associated with using our app. The anonymized usage data produced by Google Cloud Run for our API backend is securely stored and protected by Google's data security protocols.

7. Your Data Rights

As we do not collect any personal data, there is no personal data for you to request, update, or delete. If this changes, we will update this policy accordingly.

8. Changes to this Privacy Policy

We reserve the right to update this Privacy Policy at any time. We recommend you review the policy from time to time as long as you continue to use the GopherMaps app.

9. Contact Us

If you have any questions or concerns about this Privacy Policy, please contact us at coding@umn.edu

GopherMaps is a student project and is not affiliated with the University of Minnesota itself
